# Security Policy

If you discover a security issue in Frontier Graph, please do not open a public issue with exploit details.

Instead, email:

- `prashant.garg@imperial.ac.uk`

Please include:

- a short description of the issue
- affected files or surfaces
- steps to reproduce, if relevant
- any suggested mitigation

This repository is a public research project, not a managed enterprise service, so response times may vary. Still, responsible disclosure is appreciated and will be taken seriously.

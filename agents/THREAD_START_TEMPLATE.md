# Thread Start Template

Use a new thread message along these lines:

```text
Please resume FrontierGraph from the current canonical handoff pack in:

- agents/LATEST.md
- agents/PROJECT_STATE.md
- agents/NEXT_STEPS.md

Treat those files as the source of truth.

Current defaults:
- product = concept graph, not JEL
- default ontology = Baseline
- default mapping = Exploratory
- default ranking surface = suppressed baseline top-100k

Before changing anything, summarize:
1. where the project is,
2. what the immediate next task is,
3. what files you will inspect or change.
```

## If The Task Is More Specific
Append one line like:
- “Focus on website/app UX only.”
- “Focus on retrieval/corpus only.”
- “Focus on ontology compare outputs only.”
- “Focus on deployment/live state only.”

## Decision Log
- This template is intentionally short.
- It should keep a new thread from wasting tokens reconstructing history.

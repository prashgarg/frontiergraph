# Support

## Where to ask what

- **Bug report:** open a GitHub issue
- **Unclear suggestion or missing literature in the public product:** open a GitHub issue
- **Docs or repo improvement:** open a GitHub issue
- **Security issue:** email `prashant.garg@imperial.ac.uk`

## What to include

Helpful reports usually include:

- what page, script, or workflow you were using
- what you expected
- what happened instead
- any relevant screenshots, logs, or example questions/topics

## Project posture

Frontier Graph is a public beta and an active research codebase. The public site, Explorer, release bundle, and paper are the main maintained surfaces. Some exploratory research code is included for transparency and reproducibility, but not every script is polished to the same level.
